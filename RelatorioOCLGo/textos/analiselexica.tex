\subsection{Ferramenta JFlex}
A ferramenta JFlex gera analisadores léxicos em Java a partir da
especificação de expressões regulares e trechos de código associados a cada expressão. A especificação referida é escrita utilizando a linguaguem JFlex (um clone da linguagem lex), que tem como extensão .flex.
